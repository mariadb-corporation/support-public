# SUPPORT-PUBLIC

Scripts by Mariadb Support.

The following scripts are provided in this repository:

### columnstore_review
Purpose: Run on columnstore instances to gather information. Can be used to test an instance or a cluster.

### innochecksum_entire_datadir
Verify all innodb files in a data directory.


### mariadb_quick_review
A comprehensive script for gathering information about performance on a Mariadb instance.


### mariadb_xa_settle
Commit or rollback any stuck XA transactions.


### mdb-support
A basic overview of Mariadb server for support tickets.


###  operator-support
A wrapper to make it easy to download the latest Kubernetes Operator Support script.


### verify_sst_auth
THis script is useful for verifying the configuration for mariabackup SST configurations.
